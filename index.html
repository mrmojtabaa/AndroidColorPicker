AndroidColorPicker

Using Color picker

Include this project in libraries of your project and write this code in your activity

   ParangColorPicker pcp = new ParangColorPicker(MainActivity.this,false,
    new ParangColorPicker.ParangColorPickerListener() {

                        @Override
                        public void onCancel() {

                        }

                        @Override
                        public void onOk(int color) {
                                                        // your selected color is here
                        }
                    });
pcp.show(Color.BLUE);
